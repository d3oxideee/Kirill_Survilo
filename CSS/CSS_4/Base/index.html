Форматы файлов

.DOC или .doc (от англ.document)- расширение имени файла, испольщуемое для файлов,
представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для
обозначения простых текстовых файлов без форматирования, однако позже стало
использоваться для двоичных форматов с разметкой.

Portable Document Format(PDF)- Межплатформенный фомат электронных документов,
разработанный фирмой Adobe Systems с использованием ряда возможностей
языка
Документ PDF
ZIP - Популярный формат сжатия данных и архивации файлов. Файл в этом формате
обычно имеет расширение zip и хранит в сжатом или не сжатом виде один или несколько
файлов, которые можно из него извлеч путем распаковки с помощью специальной
программы. Архив
